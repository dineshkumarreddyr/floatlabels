Floating labels - An angular component

Basic Float labels (Float label = input place holder)

For Developers: Required attributes

float-type="basic"

Custom Float labels (Float label = Custom text; When watermark and float label are different)

For Developers: Required attributes

float-type="custom"

custom-label='User Name'

Position Float labels (default:left)

For Developers: Required attributes

float-type="basic"

placeholder-position-right="true"

Validation Labels (Required - Works when move out focus from control)

For Developers: Required attributes

float-type="basic"

validation-label="true"
